# TeacherScript — сценарии занятий

Краткие сценарии для преподавателя. Тренажёр: `/stakan-lenta`.

> Учебное пособие. Не инвестиционная рекомендация. Данные — mock.

---

## 10 минут — «Анатомия стакана»

**Режим:** `?mode=explain&lesson=anatomy` → затем `presenter` для заметок.

**Цель:** показать зоны терминала без торговли.

| Мин | Шаг | Что сказать | Действие |
|-----|-----|-------------|----------|
| 0–1 | top-menu | «Меню — карта зон, не кнопки ордеров» | Подсветка topbar |
| 1–2 | instrument-gazp | «Один тикер, один фокус — GAZP учебный» | — |
| 2–4 | clusters | «Объём по цене и времени; рамка POC — где был бой» | Навести на 101,75 |
| 4–5 | tape | «Зелёный = покупатель бьёт ask; красный = sell по bid» | — |
| 5–7 | orderbook / ask / bid | «Ask сверху, bid снизу; best — активные строки» | — |
| 7–8 | spread | «Линия между best bid и ask — где цена сейчас» | — |
| 8–9 | working-volume | «Рабочий объём задаёт размер клика» | Нажать `2` → 100 |
| 9–10 | hotkeys | «1–5 объём, F снять лимиты, T/Y market» | Показать без клика в стакан |

**Закрытие:** «Стакан — карта ликвидности; решение только после ленты и контекста».

## 20 минут — «Первая сделка»

**Режим:** `?mode=explain&lesson=first-trade`, шаги с `nextOnAction`.

**Цель:** одна учебная сделка limit → market → закрытие.

| Мин | Шаг | Действие ученика | Комментарий |
|-----|-----|------------------|-------------|
| 0–2 | working-volume | `1` → объём 50 | «Малый объём на первом шаге» |
| 2–5 | limit-buy | ЛКМ bid ниже spread | «Лимит встал в очередь» |
| 5–7 | hotkeys | Объяснить F / D | Не нажимать F пока |
| 7–10 | market-buy | `T` или клик best ask | «Market забрал ask — принт в ленте» |
| 10–13 | instrument-gazp | Показать позицию в шапке | «Фиксируем avg и PnL учебный» |
| 13–16 | tape | «Подтверждение агрессии после входа» | — |
| 16–18 | pre-entry | «Перед входом: объём + уровень + лента» | — |
| 18–20 | — | `D` закрыть | «Выход по правилам, не по эмоции» |

**Альтернатива:** `?mode=practice` — те же механики, автопроверка.

---

## 30 минут — «Плотность, лента, кластеры»

**Часть 1 (10 мин) — теория на туре:** `lesson=anatomy`, шаги clusters → tape → spread.

**Часть 2 (15 мин) — сценарии:** `?mode=scenario`

| Сценарий | Время | Акцент |
|----------|-------|--------|
| Плотность в bid | 3 мин | Bid 101,60–62, замедление sell-принтов |
| Выкуп ask | 3 мин | Серия buy, ask тает, best ask вверх |
| Абсорбция | 3 мин | Buy в ленте, цена стоит, кластер растёт |
| Ложный пробой | 3 мин | +2 тика и возврат |
| Пустой стакан | 3 мин | Market проскальзывает |

Управление: **Старт / Пауза / Шаг / 0.5x–2x / Скрыть пояснения**.

**Часть 3 (5 мин) — практика:** `?mode=practice`, задания на объём, лимит, ленту, абсорбцию.

**Закрытие (1 мин):**

- Плотность ≠ гарантия отбоя.
- Лента — факт; кластер — накопление.
- Market в тонком стакане — риск проскальзывания (учебный mock).
