# Slide Script Template

## Слайд: заголовок + 3 bullet

```
[Заголовок — глагол + объект]

• [Факт / правило — 8–12 слов]
• [Пример — тикер или ситуация]
• [Действие — что делать в терминале]
```

## Слайд: схема (текст для Figma/PPT)

```
[Слева]  Привод          [Центр]  Стакан + лента          [Справа]  График
         ↓                        ↓                              ↓
    Заявки, PnL              Bid/Ask, плотности              Контекст дня
```
