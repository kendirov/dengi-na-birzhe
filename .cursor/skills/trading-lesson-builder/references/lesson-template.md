# Шаблон урока

```markdown
# [Название урока]

**Цель:** ...

## 1. Контекст (2–3 мин)
- Зачем это трейдеру внутри дня

## 2. Теория (минимум)
- Только то, что нужно для практики

## 3. Разбор в терминале
- Стакан / лента / график — что смотреть

## 4. Сценарий
- Пошагово: до входа → вход → выход

## 5. Практика
- [ ] Упражнение 1
- [ ] Упражнение 2

## 6. Ошибки новичков
1. ...
2. ...

## 7. Домашнее задание
- Запись экрана / дневник / 3 наблюдения

## Связь со скринером
- Какой режим / фильтр использовать
```
