# Структура рыночного брифинга

```
# Брифинг — [дата, время MSK]

## Контекст
- Сессия: открытие / середина / закрытие
- Индекс MOEX: уровень, % дня

## Уже отыграли
- ...

## Драйверы вперёд
- ...

## Внешний фон
- USD/RUB, нефть, газ, золото, US/EU futures

## Акции — сильная реакция
| Тикер | Move | Объём | Комментарий (факт) |

## Акции — у уровней
| Тикер | Уровень | Контекст |

## Сценарии (не прогнозы)
1. Если ... → смотрим ...
2. Если ... → риск ...

## Стакан / лента
- На что обратить внимание
- Плотности / спред / аномалии

## Риски дня
- ...

## Что НЕ делать
- ...
```

Объём: 1–2 экрана текста, не простыня.
