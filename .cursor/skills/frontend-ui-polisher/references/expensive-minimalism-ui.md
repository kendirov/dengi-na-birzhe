# Expensive Minimalism UI

## Do

- Dark bg `#05070d`, cards `#06080c`
- Thin borders `border-terminal-border/60`
- Cyan for primary, green for positive, amber for cost/spread
- Compact padding: p-2.5–3 on panels
- Monospace for numbers (font-mono)

## Don't

- White cards on dark app
- Large hero sections on screener
- Excessive glow / glass
- Marketing gradients
- Icons without function

## Density

- Screener top block < 360px before table
- Inspector 320–340px max
- Lessons strip ~80px height
