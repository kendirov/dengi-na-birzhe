# Presentation & Diagram Generator

## Выход

- Текстовые блоки для слайда
- ASCII / Mermaid горизонтальные схемы
- Карточки (3–5 на экран)
- Подписи для Figma frames
- Версии: «для слайда» / «для документа»

## Стиль

См. `references/diagram-style.md`, `references/slide-blocks.md`

- Тёмный фон `#05070d`
- Акценты: cyan `#22d3ee`, green `#34d399`, amber `#fbbf24`
- Высокая информативность, без декоративного мусора
- Glassmorphism — максимум 1 элемент на слайд

## Workflow

1. Одна мысль = один слайд
2. Заголовок = действие или вывод
3. Схема ≤ 5 блоков
4. Если нужен визуал — описание для Figma MCP

## Figma MCP

После генерации текста — передать в Figma через MCP или `/add-plugin figma`
