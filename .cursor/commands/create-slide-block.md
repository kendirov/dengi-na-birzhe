# Command: create-slide-block

Создай слайд-блок или схему.

## Input

Тема, аудитория (урок / презентация / Telegram), формат.

## Steps

1. Skill: `presentation-diagram-generator`
2. One idea per slide
3. Diagram style: dark premium
4. Optional Figma MCP handoff spec

## Output formats

- **Slide text** — title + 3 bullets
- **Diagram** — ASCII or Mermaid
- **Figma spec** — frame size, colors, component names
- **Speaker notes** — 1–2 sentences

## Examples

«Схема рабочего места трейдера»
«Блок: отскок от плотности»
«Карточки: риск на сделку / размер / лимит / цена ошибки»
