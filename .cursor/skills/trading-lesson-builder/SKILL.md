# Trading Lesson Builder

## Вход

- Сырой текст, транскрипт, идея, скрин терминала
- Существующий урок в `app/lesson/`

## Выход

| Блок | Описание |
|------|----------|
| Цель урока | Одно предложение |
| Структура | 3–7 секций |
| Объяснение | Простой язык, без воды |
| Пример на рынке | Конкретный тикер/сценарий |
| Блок для слайда | 3–5 bullet |
| Практика | Упражнение в терминале |
| Чеклист ученика | 5–8 пунктов |
| Ошибки новичков | 3–5 типичных |
| Домашнее задание | Измеримое |

## Шаблоны

- `references/lesson-template.md`
- `references/slide-script-template.md`

## Стиль

- CScalp / привод / стакан / лента / кластера
- Взрослый трейдинг, без инфоцыганства
- Связь с курсом: https://schoollive.ru/chellendzh-dengi-na-birzhe

## Компоненты проекта

- `LessonShell`, `LessonSection`, `StepTimeline`
- `OrderBookSandbox`, `DensityScenario`
- `CourseLessonsStrip` — навигация занятий

## Не делать

- Огромные текстовые блоки на скринере
- Новые интерактивы без явной задачи
