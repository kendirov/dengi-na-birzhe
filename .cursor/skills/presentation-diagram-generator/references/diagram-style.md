# Diagram Style — Expensive Minimalism

## Цвета

| Role | Hex | Use |
|------|-----|-----|
| Background | `#05070d` | Slide bg |
| Surface | `#06080c` | Cards |
| Border | `#141c2a` | Lines |
| Text | `#e2e8f0` | Body |
| Muted | `#64748b` | Labels |
| Cyan | `#22d3ee` | Primary action |
| Green | `#34d399` | Positive / bid |
| Amber | `#fbbf24` | Warning / ask |
| Red | `#f87171` | Risk |

## Типографика

- Заголовок: 20–24px semibold
- Body: 14–16px
- Labels: 10–11px uppercase tracking

## Схемы

```
┌─────────┐     ┌─────────┐     ┌─────────┐
│ Block A │ ──► │ Block B │ ──► │ Block C │
└─────────┘     └─────────┘     └─────────┘
```

Или Mermaid `flowchart LR` для сложных потоков.

## Метафоры

- Стакан = «стена» bid/ask
- Плотность = «магнит» уровня
- Лента = «агрессия» рынка
